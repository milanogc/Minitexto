Minitexto
=========

Simple project to demonstrate the use of Ember.js with (Java) Spring Framework without XML configuration.

Client side
-----------

- Ember
- Ember Data
- Twitter Bootstrap

Server side
-----------

- Maven
- JPA 2.0 (Hibernate/H2)
- Spring Data
- Spring MVC
- Jackson

Run the project
---------------

    mvn tomcat7:run
